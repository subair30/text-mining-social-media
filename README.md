# Text Mining for Social Media Insights

## Description
This project demonstrates text mining techniques applied to social media data.

## Features
- Named Entity Recognition (NER)
- Relation Extraction with Graph Visualization
- Feature Extraction (BoW, TF-IDF)
- Sentiment & Spam Classification
- Data Visualization (Heatmap, Charts)

## Technologies Used
- Python
- Scikit-learn
- spaCy
- NetworkX
- Matplotlib

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run the code:
   python main.py

## Output
- Entity Relationship Graph
- Feature Extraction Tables
- Accuracy Graph
- Confusion Matrix
